Cardiovascular imaging research software

HAGRad Viewer

A local DICOM research environment for cardiac CT review, MPR, structured measurements, image-quality analysis, Noise Power, EAT assessment, and quantitative coronary angiography workflows.

Download HAGRad Explore workflows

Research use only. Not for clinical diagnosis or patient care.

HAGRad is designed for retrospective scientific image analysis. DICOM processing runs locally on the user's computer, so research datasets do not need to be uploaded to a public website.

Suite overview

One platform, multiple research workflows

HAGRad keeps the core viewer familiar while separating specialized measurements into focused modules. Each workflow exports structured files for downstream analysis.

Core

HAGRad Viewer

Local DICOM loading, window/level, zoom, pan, MPR, vessel profile, blooming/stenosis diameter tools, and ZIP exports.

CTA

Image Quality

Objective coronary CTA ROI metrics, subjective reader scoring, customizable protocols, and structured export.

Physics

Noise Power

CT phantom square ROI statistics, circular NPS placement, multi-reconstruction NPS comparison, TTF metrics, and export bundles.

CT

EAT

Epicardial adipose tissue contouring, threshold review, multi-reconstruction comparison, and CSV/PNG reporting.

XA

QCA

Invasive angiography frame selection, vessel segmentation workflow, stenosis measurements, and export bundles.

Local installation

Download HAGRad Viewer

The website is the public portal. HAGRad Viewer itself runs locally, keeps DICOM files on the user's computer, and opens from one app icon after download.

macOS

HAGRad for Mac

Download file: HAGRad-Viewer-macOS.dmg

  1. Open the DMG.
  2. Double-click HAGRad Viewer.app.
  3. First launch only: if macOS says the developer cannot be verified, Control-click the app, choose Open, then confirm Open.

No Python, Terminal, OpenSSL, or Apple Command Line Tools are required for normal use.

Download for macOS (.dmg)

Windows

HAGRad for Windows

Download file: HAGRad-Viewer-Windows.zip

  1. Unzip the downloaded file.
  2. Double-click HAGRad Viewer.exe.
  3. First launch only: if Windows SmartScreen appears, choose More info, then Run anyway if you downloaded it from this page.

The executable starts the local server, opens the browser, and uses localhost HTTP when HTTPS certificates are unavailable.

Documentation

Built for transparent research exports

HAGRad exports measurement tables, image panels, metadata-aware CSV files, and ZIP bundles designed for statistics, manuscript figures, and reproducible study organization.

Local by design

DICOM files are loaded and processed on the user's own computer.

Structured output

Exports are designed for Excel, R, Python, GraphPad, and research archives.

Workflow-specific tools

Image Quality, EAT, and QCA remain separate enough to avoid measurement-state confusion.
