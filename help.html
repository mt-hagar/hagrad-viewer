HAGRad Viewer

This local browser viewer loads DICOM slices, creates a presentation stack, and renders synchronized linked comparison views.

Controls

  • Mouse wheel: scroll slices
  • Arrow keys / Page Up / Page Down: scroll the focused viewport
  • Space: start or pause cine in the presentation viewport
  • Toolbar buttons: switch between WW/WL, pan, zoom, length, probe, and ROI tools
  • Type exact window width and level values in the sidebar inputs
  • Export the current view, a 4-up panel, or a cine clip for PowerPoint
  • `M`: toggle between presentation mode and 4-up linked mode

Important note

This add-in is intended for presentation and teaching workflows. It is not a validated diagnostic workstation.
