In `Contour`, click and drag to trace a closed loop. In `Adjust`, drag a handle to refine the contour or drag inside the contour to translate it. In `Rubber`, brush over unwanted pixels to exclude them from the EAT count.

`Pan (M)` moves the image. `Zoom (Z)` lets you zoom with drag or mouse wheel. `Let's Go Segment` uses your existing contoured slices as anchors, while `Automatic EAT (A)` tries an image-only first pass across the slab. Both leave every slice editable afterward.

`Model Assist (I)` uses the local Python backend and a trained heart-localization model when available, then imports editable pericardial starting contours back into this review workflow.

`Store Reference (K)` stores the active reconstruction and your confirmed manual contours as a local reference bundle, then updates the local contour profile used by future model-assisted runs.

More shortcuts: `[` top, `]` bottom, `F` full stack, `,`/`.` copy prev/next, `O` overlay, `V` compare, `T` transfer, `X` clear contour, `Shift+X` clear rubber, `A` for Automatic EAT, `I` for Model Assist, and `K` for Store Reference.
