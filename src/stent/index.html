`ROI Circle` starts from the initial click as the center. Drag outward to size it, release to create it, then move it or drag the single outer handle to resize it.

`ROI Draw` traces freehand while you hold the left mouse button. `ROI Multiple Click` places discrete points and finishes on double-click.

`Adjust ROI` lets you redraw part of an existing ROI contour and blends the corrected segment in.

`W` windows, `Z` zooms, `M` pans. Use `Edit` for existing objects, and use the profile tools only after the vessel path looks right.

MPR Sync

Shared across reconstructions

`MPR Coord` lets you drag the shared crosshair center. Rotation still comes from the three angle controls below it.

You can also grab a colored MPR line directly in the viewport to rotate that plane with the mouse. The current viewport stays fixed while the other MPR projections rotate.

`Stent Line` and `Stent Square` quantify the stent-lumen interface. `Calcified Line` quantifies plaque-lumen blooming from a line across contrast lumen and calcified plaque.

`Stent Path` is manual. Build the centerline with left-click axial anchors, then use the cross-section panel below the viewer to adjust outer and inner diameters for the active sample.

Use `Edit` to click an existing annotation, move it, drag its handles, double-click a text label to rename it, or press `Delete`.

The profile panel switches between stent-lumen metrics and calcified plaque-lumen blooming metrics depending on the selected profile.

Controls

Mouse wheel scrolls slices. Click a viewport to focus it. Arrow keys scroll the focused viewport. Space starts cine in the presentation stack.

Hold right click in the axial presentation viewer and drag to scroll very quickly through the stack.

In 4-up mode, the viewer shows real axial, coronal, and sagittal reformats. `Arrow` draws a pointer, `Text Label` lets you place a short note, and `ROI` closes on double-click.

Shortcuts: `r` ROI, `c` picture export, `v` cine export, `m` pan, `z` zoom, `w` window, `1-4` presets.

Extra shortcuts: `g` stent path, `q` line profile, `s` square profile, `x` MPR coordinate tool.

`x` switches to `MPR Coord` so you can reposition the shared crosshair directly in any viewport.

Loaded reconstructions share one MPR coordinate system, so switching reconstructions keeps the same position, rotation, and zoom.

`Load DICOM Files` lets you select normal `.dcm` files. `Load Folder` is there if you want to pick one whole patient folder at once.
